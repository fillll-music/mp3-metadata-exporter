MP3 Metadata Exporter - User Guide

Version 1.0.0

What is MP3 Metadata Exporter?

MP3 Metadata Exporter is a desktop application that extracts ID3 tag metadata from MP3 files and exports it to JSON format. This is useful for creating music library databases, analyzing your music collection, or preparing data for music scheduling applications.

Features

  • Extract metadata from MP3 files (artist, album, title, genre, year, duration)
  • Preview metadata with configurable sample sizes (1-100 tracks)
  • Export to JSON format with automatic file saving
  • Handle files with missing or incomplete metadata
  • Fallback to filename parsing when tags are missing
  • Progress tracking for large libraries
  • Configurable output directory settings
  • Settings persistence between sessions

How to Use

Step 1: Configure Settings (Optional)

  1. Click the gear icon (⚙️) next to the Export JSON button
  2. Set the number of tracks to preview (1-100, default: 10)
  3. Choose an output directory for JSON files (optional)
  4. Click Save to apply your settings

Step 2: Preview Your Library

  1. Click the "Preview Library" button
  2. Select the folder containing your MP3 files
  3. Wait for the scan to complete (only processes the configured number of tracks)
  4. Review the metadata in the preview table

Step 3: Export to JSON

  1. Click the "Export JSON" button
  2. If you haven't selected a folder yet, choose the folder containing your MP3 files
  3. The app will automatically save the JSON file to your configured output directory
  4. Wait for the export to complete (processes all MP3 files in the folder)

Settings

Tracks to Preview

This setting controls how many tracks are shown in the preview table. The app randomly samples this many tracks from your library for quick previewing. This makes previews fast even for large libraries.

Output Directory

Set your preferred directory for saving JSON export files. If left empty, the app will save the JSON file in the same folder as your MP3 files.

Output Format

The exported JSON file contains an array of track objects with the following structure:

[
  {
    "path": "filename.mp3",
    "title": "Song Title",
    "artist": "Artist Name",
    "album": "Album Name",
    "genre": ["Rock", "Alternative"],
    "year": 2020,
    "duration": 180.5
  }
]

Tips

Large Libraries: For folders with many MP3 files, the export process may take some time. The progress bar will show you the current status. Preview mode is much faster as it only processes a sample of your files.
Missing Metadata: If an MP3 file has missing ID3 tags, the app will attempt to extract information from the filename using common patterns like "Artist - Title" or "Artist - Album - Title".
File Organization: For best results, organize your MP3 files in folders by artist or album before exporting.
Settings Persistence: Your settings are automatically saved and will be remembered the next time you open the app.

Supported File Formats

  • MP3 files (.mp3)
  • ID3v1 and ID3v2 tags

System Requirements

  • Windows 10 or later
  • macOS 10.14 or later
  • Linux (Ubuntu 18.04 or later)
  • Minimum 4GB RAM
  • 100MB free disk space
